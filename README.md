1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the tool:**
    You can use `ts-node` to run the tool directly.

    **Ingest a document:**

    ```bash
    node src/rag.js
    ```

    **Query your documents:**

    ```bash
    npx ts-node src/index.ts query "What is MongoDB?"
    ```
